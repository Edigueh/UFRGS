## Keywords

### Classes & Objects

| Keyword | Description |
| --------- | ------------- |
| `class` | Defines a class |
| `struct` | Defines a structure |
| `enum` | Declares an enumerated type |
| `this` | A pointer to a class instance available inside class methods and constructors |
| `new` | Reserves dynamic memory |
| `delete` | Frees dynamic memory |
| `friend` | Specifies classes and functions which have access to private and protected members |
| `virtual` | Specifies that a method in a base class can be overridden in child classes |
| `template` | Declares a template class or template function |

### Exception Handling

| Keyword | Description |
| --------- | ------------- |
| `try` | Creates a try...catch statement |
| `catch` | Catches exceptions generated by try statements |
| `throw` | Creates a custom error which can be caught by a try...catch statement |

### Other Keywords

| Keyword | Description |
| --------- | ------------- |
| `auto` | Automatically detects the type of a variable based on the value you assign to it |
| `const` | Defines a variable or parameter as a constant (unchangeable) |
| `static` | Specifies that a variable in a function keeps its value after the function ends |
| `typedef` | Defines a custom data type |
| `using` | Allows variables and functions from a namespace to be used without the namespace's prefix |
| `namespace` | Declares a namespace |
| `sizeof` | Returns the size of a data type or variable |
| `explicit` | Prevents implicit type conversions |
| `extern` | Declares a variable or function that is defined in another file |
| `inline` | Suggests to the compiler to insert the function's code at the call site |
| `mutable` | Allows a member of an object to be modified even if the object is const |
| `operator` | Defines or overloads an operator |
| `register` | Suggests storing a variable in a CPU register |
| `volatile` | Indicates that a variable's value may change unexpectedly |

### Alternative Operators

| Keyword | Description |
| --------- | ------------- |
| `and` | An alternative way to write the logical `&&` operator |
| `and_eq` | An alternative way to write the `&=` assignment operator |
| `bitand` | An alternative way to write the `&` bitwise operator |
| `bitor` | An alternative way to write the `|` bitwise operator |
| `compl` | An alternative way to write the `~` bitwise operator |
| `not` | An alternative way to write the logical `!` operator |
| `not_eq` | An alternative way to write the `!=` comparison operator |
| `or` | An alternative way to write the logical `||` operator |
| `or_eq` | An alternative way to write the `|=` assignment operator |
| `xor` | An alternative way to write the `^` bitwise operator |
| `xor_eq` | An alternative way to write the `^=` assignment operator |

### Special Values

| Keyword | Description |
| --------- | ------------- |
| `true` | A boolean value equivalent to 1 |
| `false` | A boolean value equivalent to 0 |
| `nullptr` | Represents a null pointer |

## Data Types

### Primitive Types

```cpp
// Integer types
int x = 5;              // 32-bit integer (-2,147,483,648 to 2,147,483,647)
short s = 10;           // 16-bit integer (-32,768 to 32,767)
long l = 100000L;       // 64-bit integer
long long ll = 1LL;     // 64-bit integer

// Unsigned variants
unsigned int u = 5;     // Only positive values (0 to 4,294,967,295)
unsigned long ul = 100; // Larger positive range

// Floating point
float f = 3.14f;        // 32-bit floating point (6-7 decimal digits)
double d = 3.14159;     // 64-bit floating point (15 decimal digits)
long double ld = 3.14L; // Extended precision

// Character and Boolean
char c = 'A';           // Single character
bool flag = true;       // Boolean (true or false)

// Special
void* ptr;              // Generic pointer
auto var = 10;          // Type deduced by compiler
```

### Type Modifiers

```cpp
const int CONSTANT = 100;        // Cannot be modified
static int counter = 0;          // Retains value between function calls
extern int globalVar;            // Defined in another file
volatile int sensorValue;        // May change unexpectedly
```

## STL Containers

### Vector (Dynamic Array)

```cpp
#include <vector>
vector<int> v;           // Empty vector
vector<int> v(n);        // Vector of size n
vector<int> v(n, val);   // Vector of size n with value val

// Operations
v.push_back(x);          // Add element to end - O(1)
v.pop_back();            // Remove last element - O(1)
v.size();                // Number of elements
v.empty();               // Check if empty
v[i];                    // Access element (no bounds check)
v.at(i);                 // Access element (with bounds check)
v.front();               // First element
v.back();                // Last element
v.clear();               // Remove all elements
v.erase(v.begin() + i);  // Remove element at index i
v.insert(v.begin() + i, val); // Insert at index i
```

### Set (Unique Sorted Elements)

```cpp
#include <set>
set<int> s;              // Empty set

// Operations
s.insert(x);             // Add element - O(log n)
s.erase(x);              // Remove element - O(log n)
s.find(x);               // Find element - O(log n)
s.count(x);              // Check if exists (0 or 1) - O(log n)
s.size();                // Number of elements
s.empty();               // Check if empty
s.clear();               // Remove all elements
```

### Map (Key-Value Pairs)

```cpp
#include <map>
map<string, int> m;      // Empty map

// Operations
m[key] = value;          // Insert/update - O(log n)
m.insert({key, value});  // Insert pair
m.erase(key);            // Remove by key - O(log n)
m.find(key);             // Find by key - O(log n)
m.count(key);            // Check if exists (0 or 1)
m.size();                // Number of pairs
m.empty();               // Check if empty
m.clear();               // Remove all pairs

// Iteration
for (auto& [key, value] : m) {
    // code
}
```

### Queue (FIFO)

```cpp
#include <queue>
queue<int> q;            // Empty queue

// Operations
q.push(x);               // Add to back - O(1)
q.pop();                 // Remove from front - O(1)
q.front();               // Access front element
q.back();                // Access back element
q.size();                // Number of elements
q.empty();               // Check if empty
```

### Stack (LIFO)

```cpp
#include <stack>
stack<int> st;           // Empty stack

// Operations
st.push(x);              // Add to top - O(1)
st.pop();                // Remove from top - O(1)
st.top();                // Access top element
st.size();               // Number of elements
st.empty();              // Check if empty
```

### Priority Queue (Max Heap by default)

```cpp
#include <queue>
priority_queue<int> pq;  // Max heap

// Min heap
priority_queue<int, vector<int>, greater<int>> minHeap;

// Operations
pq.push(x);              // Add element - O(log n)
pq.pop();                // Remove top element - O(log n)
pq.top();                // Access top element - O(1)
pq.size();               // Number of elements
pq.empty();              // Check if empty
```

### Pair

```cpp
#include <utility>
pair<int, int> p = {1, 2};
pair<int, string> p2 = make_pair(1, "hello");

// Access
p.first;                 // First element
p.second;                // Second element
```

### Deque (Double-ended Queue)

```cpp
#include <deque>
deque<int> dq;           // Empty deque

// Operations
dq.push_back(x);         // Add to back - O(1)
dq.push_front(x);        // Add to front - O(1)
dq.pop_back();           // Remove from back - O(1)
dq.pop_front();          // Remove from front - O(1)
dq.front();              // Access front
dq.back();               // Access back
dq[i];                   // Random access
```

## Algorithms

### Sorting

```cpp
#include <algorithm>

// Sort in ascending order
sort(v.begin(), v.end());

// Sort in descending order
sort(v.begin(), v.end(), greater<int>());

// Sort with custom comparator
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // Descending
});

// Partial sort
partial_sort(v.begin(), v.begin() + k, v.end());

// Check if sorted
is_sorted(v.begin(), v.end());
```

### Searching

```cpp
// Binary search (requires sorted array)
binary_search(v.begin(), v.end(), value);  // Returns true/false

// Lower bound (first element >= value)
auto it = lower_bound(v.begin(), v.end(), value);

// Upper bound (first element > value)
auto it = upper_bound(v.begin(), v.end(), value);

// Find element
auto it = find(v.begin(), v.end(), value);

// Count occurrences
int cnt = count(v.begin(), v.end(), value);
```

### Min/Max

```cpp
// Minimum and maximum element
auto minIt = min_element(v.begin(), v.end());
auto maxIt = max_element(v.begin(), v.end());

// Min/max of two values
int minVal = min(a, b);
int maxVal = max(a, b);

// Min/max of multiple values
int minVal = min({a, b, c, d});
```

### Permutations

```cpp
// Next permutation
next_permutation(v.begin(), v.end());

// Previous permutation
prev_permutation(v.begin(), v.end());
```

### Reverse and Rotate

```cpp
// Reverse
reverse(v.begin(), v.end());

// Rotate
rotate(v.begin(), v.begin() + k, v.end());
```

### Unique and Remove

```cpp
// Remove duplicates (requires sorted array)
auto last = unique(v.begin(), v.end());
v.erase(last, v.end());

// Remove specific value
v.erase(remove(v.begin(), v.end(), value), v.end());
```

### Accumulate and Other Numeric Operations

```cpp
#include <numeric>

// Sum of elements
int sum = accumulate(v.begin(), v.end(), 0);

// Product of elements
int product = accumulate(v.begin(), v.end(), 1, multiplies<int>());

// GCD and LCM (C++17)
int g = gcd(a, b);
int l = lcm(a, b);

// Partial sum
partial_sum(v.begin(), v.end(), result.begin());
```

### Fill and Generate

```cpp
// Fill with value
fill(v.begin(), v.end(), value);

// Generate values
generate(v.begin(), v.end(), rand);
```

## Strings

### String Operations

```cpp
#include <string>

string s = "Hello";
string s2 = "World";

// Concatenation
string s3 = s + " " + s2;

// Length
s.length();              // or s.size()

// Access characters
s[0];                    // First character
s.at(0);                 // With bounds check
s.front();               // First character
s.back();                // Last character

// Modify
s.push_back('!');        // Add character to end
s.pop_back();            // Remove last character
s.append(" World");      // Append string
s.insert(5, " there");   // Insert at position

// Substring
s.substr(start, length); // Extract substring
s.substr(start);         // From start to end

// Find
s.find("ll");            // Find substring (returns position or string::npos)
s.rfind("l");            // Find last occurrence

// Replace
s.replace(start, length, "new");

// Compare
s.compare(s2);           // Returns 0 if equal, <0 if less, >0 if greater
s == s2;                 // Boolean comparison

// Case conversion (requires <algorithm>)
transform(s.begin(), s.end(), s.begin(), ::toupper);
transform(s.begin(), s.end(), s.begin(), ::tolower);

// Clear
s.clear();
s.empty();               // Check if empty
```

### C-Style String Functions

```cpp
#include <cstring>

char str1[100] = "Hello";
char str2[100] = "World";

strlen(str1);            // Length
strcpy(str1, str2);      // Copy
strcat(str1, str2);      // Concatenate
strcmp(str1, str2);      // Compare
strchr(str1, 'e');       // Find character
strstr(str1, "ll");      // Find substring
```

```cpp
log10(x);                // Base 10 logarithm

exp(x);                  // e^x

sinh(x);                 // Hyperbolic sine
cosh(x);                 // Hyperbolic cosine

```
